# Project NEON BASTION - 식별성 강화 이미지 프롬프트

기존 버전에서 타워와 몬스터가 모두 비슷한 기계 질감과 비슷한 색조로 보여 구분이 약했던 문제를 해결하기 위해, 이번 버전은 **실루엣 우선 + 대면적 색상 블로킹 + 역할별 형태 분리**를 기준으로 다시 작성했다.

사용 기준: ChatGPT에서 한 개체씩 생성하고, 이후 배경 제거 후처리를 진행한다. 1:1 비율 권장.

## 이번 버전의 핵심 규칙

- 각 개체는 **대표 실루엣 1개**, **대표 색상군 1개**, **다른 개체와 겹치지 않는 형태 언어**를 가진다.
- 기존처럼 회색 금속 위에 작은 네온선만 넣지 말고, **큰 면적의 색상 패널**이 보이게 한다.
- 썸네일이나 카드 크기에서도 바로 식별되도록, 모양과 색을 동시에 분리한다.
- 프롬프트 안에 **겹치면 안 되는 다른 개체 타입**을 함께 적어, 모델이 비슷한 모양으로 수렴하는 것을 줄인다.
