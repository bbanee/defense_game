# Project NEON BASTION - 타워 특수효과 발동/피격 오버레이 이미지 프롬프트

이번 버전은 **타워별 특수 스킬 발동 이펙트 1종**과 **몬스터 피격 상태 오버레이 1종**을 한 세트로 정리했다.
사용 기준은 **ChatGPT로 이펙트 이미지를 생성한 뒤 후처리로 검정 배경 제거**하는 방식이다.

## 사용 기준

- 발동 이펙트는 **발사 / 방출 / 캐스팅 순간에 보이는 효과만 단독 생성**한다.
- 피격 오버레이는 **몬스터 본체 주변에 감기듯 표시되는 상태 이상 / 디버프 이미지**를 만든다.
- 피격 오버레이는 **바닥 장판이 아니라 몬스터 몸 둘레에 붙는 합성용 이미지**로 설계한다.
- 오버레이는 실제 몬스터를 그리지 않고, **중앙을 비워서 몬스터 스프라이트 위에 얹기 쉽게** 만드는 게 핵심이다.

## 공통 리트라이 문구

If the result includes a tower, a monster, a weapon, or turns into a floor decal or a full magic circle, retry with this sentence appended: remove all units and scenery, keep only one isolated VFX element on a pure black background, keep the center mostly open for an enemy sprite, wrap the effect around an invisible target body, and do not place it on the ground.
