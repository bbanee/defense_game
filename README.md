## UTF-8 (Windows/PowerShell)

If Korean text appears broken in terminal output, run:

```powershell
chcp 65001
[Console]::InputEncoding  = [System.Text.UTF8Encoding]::new()
[Console]::OutputEncoding = [System.Text.UTF8Encoding]::new()
```

This repository is configured for UTF-8 text files via `.editorconfig`,
`.vscode/settings.json`, and `.gitattributes`.
