# Project NEON BASTION - 타워 궁극기 이펙트 이미지 프롬프트

이번 버전은 **타워 15종 각각의 궁극기 느낌 시그니처 이펙트 1종**으로 구성했다.
사용 기준은 **ChatGPT로 이펙트 이미지를 생성한 뒤 후처리로 배경 제거**하는 방식이다.

## 사용 기준

- 모든 프롬프트는 **타워 본체 없이 효과만 단독 생성**하도록 설계했다.
- 배경은 **순검정 배경**을 전제로 해, 나중에 잘라내거나 additive / screen / mixed 계열로 합성하기 쉽도록 맞췄다.
- 이번 세트는 일반 공격 이펙트가 아니라 **각 타워의 시그니처 궁극기 연출용**이다.
- 색상, 형태, 실루엣이 타워별로 확실히 구분되도록 구성했다.
