## 메모
- 등급 분포: 일반 5 / 보통 5 / 유니크 3 / 전설 2
- 공속초: 한 번 공격 후 다음 공격까지의 간격(초)
- 궁극기 공통: 발동 시 기본 피해 배수(`ultimateDamageMultiplier`) 적용 후 추가 효과 발동
