# 타워 레벨업 필요 조각표

기준:
- 표의 수치는 `현재 레벨 -> 다음 레벨` 업그레이드에 필요한 조각 수입니다.
- 예: `Lv1->2 = 10`이면, 레벨 1 타워를 레벨 2로 올리는 데 조각 10개가 필요합니다.
- 이번 조정은 `Lv1~5`는 기존보다 조금 더 어렵게, `Lv6 이후`는 더 가파르게 증가하도록 다시 설계했습니다.
- 등급과 타워 역할에 따라 `base`와 `multiplier`를 다르게 줬습니다.
