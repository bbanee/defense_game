# Project NEON BASTION - 공격 이펙트 이미지 프롬프트

이번 버전은 **타워별 공격/지원 이펙트 2종**, **몬스터별 시그니처 공격 이펙트 1종**으로 구성했다.
사용 기준은 **ChatGPT로 이펙트 이미지를 생성한 뒤 후처리로 배경 제거**하는 방식이다.

## 사용 기준

- 이펙트는 일반 캐릭터/타워 이미지와 다르게 **무기 본체나 캐릭터 없이 효과만 단독 생성**한다.
- 배경은 **순검정 배경**을 기준으로 잡아, 후처리에서 제거하거나 additive/screen 계열 합성에 쓰기 쉽게 한다.
- 타워는 실전 제작 효율을 위해 **투사체/발사형 1장 + 충돌/영역형 1장** 기준으로 구성했다.
- 몬스터는 각 개체를 대표하는 **시그니처 공격 효과 1장** 기준으로 구성했다.

## 공통 리트라이 문구

If the result includes a tower, monster, weapon, or scenery, retry with this sentence appended: remove all units and scenery, keep only one isolated VFX element on a pure black background, make the effect shape cleaner and more readable.
