# Project NEON BASTION - 세로형 맵 이미지 프롬프트

휴대폰 **세로 모드** 기준으로 다시 정리한 맵 프롬프트 모음이다.
이번 버전은 **가로로 넓게 퍼지는 맵**이 아니라, **세로 화면 안에서 위에서 아래로 읽히는 구조**를 강하게 유도하도록 작성했다.

핵심 기준:
- 스마트폰 세로 화면에 맞는 **tall vertical composition**
- 적 진입은 **상단**, 수비 목표물은 **하단 또는 하단 1/3 지점**
- 맵 전체가 한 장 안에 보여야 함
- 경로와 타워 배치 구역이 한눈에 구분되어야 함
- 분기형 / 나선형 / 연속형이 실루엣만 봐도 다르게 보여야 함

## 공통 스타일 가이드

top-down isometric cyberpunk tower defense gameplay map concept, designed for smartphone portrait mode, tall vertical 9:16 composition, the entire map must fit inside one portrait frame, width compressed and height emphasized, clear top-to-bottom gameplay flow, enemy entrance near the top edge, defended human base or defense core near the lower third or bottom edge, gameplay-first layout, crystal-clear enemy route readability, clearly separated tower build pads and road lanes, readable at thumbnail size, strong overall silhouette, empty battlefield before combat, no characters, no monsters, no vehicles, no explosions, no muzzle flashes, no UI, no text, no logo, no watermark

## 공통 리트라이 문구

If the result looks landscape, too wide, or too cinematic instead of a readable gameplay map, retry with this sentence appended: make the map clearly portrait-oriented for a smartphone screen, extend the map vertically, reduce the width, place the entrance at the top and the defended base at the bottom or lower third, simplify decorative buildings, make the route and tower build pads much more readable, and remove cinematic clutter.

## 생성 팁

- 가능하면 **세로 비율(9:16 또는 비슷한 portrait ratio)**로 생성
- 첫 결과가 가로형이면 공통 리트라이 문구를 뒤에 붙여 재생성
- 도시 배경이 너무 많으면 'reduce decorative buildings'를 추가
- 실제 게임용이면 이후 단계에서 타일/패드/경로를 따로 단순화해 사용하는 것이 좋음
